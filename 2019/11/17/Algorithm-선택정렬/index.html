선택정렬(selection sort)

본 포스팅은 <그림으로 정리한 알고리즘과 자료구조>를 바탕으로 진행합니다.



선택정렬 알고리즘 알아보기

선택정렬(Selection Sort) 알고리즘은 가장 작은 데이터를 찾아 가장 앞 데이터와 교환하는 알고리즘으로, 사용방법은 다음과 같습니다.

  • 데이터 중에서 가장 작은 것을 찾아서 처음에 있는 것과 위치를 교환합니다.
  • 두 번째부터 마지막까지의 데이터 중에서 가장 작은 것을 찾아서 두 번째와 위치를 교환합니다.
  • 위의 과정을 계속 반복합니다.
