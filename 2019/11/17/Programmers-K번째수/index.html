시퀀스 타입

위의 코드에서 배열 array는 파이썬에 내장된 시퀀스 타입인 리스트로 구현되었다.
시퀀스 타입의 객체는 인덱스 연산을 통해 내부 항목에 접근할 수 있다.

리스트

리스트는 중복이 가능하고 순차적인 데이터를 나타낼 수 있으며, 내부 항목을 변경가능하다.

  • 정렬하기(sort, sorted)
    sort는 리스트 자체를 정렬, sorted는 리스트의 정렬 복사본을 반환한다.

  • 리스트 복사(copy)
    copy 함수, list 함수, 슬라이스 연산[:]

2. 1번에서 할당한 리스트, 튜플 변수를 이용해 다시 문자열을 만든다.
