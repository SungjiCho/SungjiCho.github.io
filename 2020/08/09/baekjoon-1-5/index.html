[BaekJoon] 입출력과 사칙연산


변수는 데이터가 저장된 주 기억장치의 특정 위치를 의미한다. 데이터를 저장하는 주 기억장치는 저장된 데이터를 구별하기 위해 서로 다른 위치에서 저장하며, 저장된 위치 정보는 주소로 표기한다. 이때 저장 위치 정보인 주소는 프로그램에서 변수에 대응한다.


위에서 사용된 변수는 총 2개이다. 이를 통해 한 프로그램 내에 여러 개의 변수가 사용되는 것을 알 수 있다. 따라서 변수를 구별할 수 있어야 하기 때문에 변수는 각각의 이름을 갖는다. 변수 이름은 다른 변수와 구별할 수 있도록 유일해야 한다.


변수는 반드시 사용하기 전에 알려야 하는데, 이를 선언이라 한다. 변수 이름 앞에 사용되는 int나 double은 변수의 데이터 유형을 나타내는 예약어로, 변수에 저장될 값이 어떤 유형인지를 나타낸다. 변수 선언을 위해 프로그램 작성자는 프로그램에서 작성해야 하는 데이터와 사용하는 컴퓨터 언어에서 제공되는 컴퓨터 표현 단위를 정확히 알아야 하고 이를 이용해서 적절한 변수를 선언해야 한다.


위의 예제 소스에서 상수 0은 변수 a에 할당된다. 즉 a를 참조하면 저장된 값 0을 사용할 수 있다. 상수 0 또한 변수 b에 할당되었다. 마지막 문장 ‘return 0’에서 상수 0이 사용된 것을 알 수 있고, 상수 0은 특정 변수에 저장된 것은 아니지만 사용되었다는 것 자체로 프로그램 실행 중 주 기억장치 어딘가에 저장된다. 프로그램을 실행하면 입력을 받는다.


프로그램을 작성한다는 것은 일련의 데이터를 처리한다는 것이다. 즉, 복잡한 계산을 하거나 많은 양의 데이터를 처리하는 등의 일을 수행하는 프로그램을 만드는 것이다. 이때 프로그램의 작업 대상은 데이터이다. 따라서 프로그램을 작성할 때 가장 먼저 해야 할 일은 대상 데이터를 저장할 변수와 변수에 저장되는 데이터의 성격을 결정하는 것이다. ‘성격’이란 데이터 특성에 따른 컴퓨터의 표현이나 저장 방식으로, 숫자인가 문자인가 또는 숫자라면 소수점이 있는 형태인가 아닌가 등을 구분하는 것이다.
